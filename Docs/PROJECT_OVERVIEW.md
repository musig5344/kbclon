## 🚀 빠른 시작 (개발 환경)

```bash
# 1. 프로젝트 디렉토리 이동
cd /home/kali/바탕화면/kb-starbanking-clone

# 2. 개발 서버 시작
npm start

# 3. 브라우저에서 http://localhost:3000 접속
```

### 🎬 데모 시연 순서
1. **앱 시작** → KB 스플래시 화면 (곰돌이 로딩)
2. **"다른 로그인 방법 선택"** → 바텀시트 표시  
3. **"아이디" 탭 선택** → 실제 로그인 폼
4. **테스트 계정으로 로그인** → 완전한 대시보드

  - 취소/확인 두 버튼 레이아웃
